# Dynamic Warehouse Solutions - Mezzanine Offices Melbourne

A React/Vite location landing page designed for the "Custom Mezzanine Office Fitouts Melbourne" campaign. Features a boutique industrial design aesthetic, soundproofing insights, project galleries, and optimised lead capture.

## Setup

1. Make sure Node.js is installed.
2. Clone this repository.
3. Install dependencies:
   ```bash
   npm install
   ```

## Development

Run the local development server:
```bash
npm run dev
```

## Deployment

This project uses **GitHub Actions** to automatically build and deploy to **GitHub Pages**. 
When code is pushed to the `main` branch, the `.github/workflows/deploy.yml` workflow triggers.

Make sure to enable GitHub Pages in your repository settings:
- Go to Settings > Pages
- Under "Build and deployment", set "Source" to **GitHub Actions**.

## Key Features

- **React & Vite**: Fast, modern frontend.
- **Scroll CTA**: Mobile-friendly CTA button sticking to the bottom on upwards scroll.
- **Form CRO**: Streamlined lead capture form focusing on conversion rate optimisation.
- **Registered Builder Banner**: Social proof integration for credibility.
